Adds a From URL tab to the media modal that fetches an image by URL and sets it as the featured image.

== Description ==
Paste any direct image URL, click Fetch and set featured... the plugin sideloads the image into your Media Library and sets it as the featured image.

== Installation ==
1. Upload the plugin ZIP via Plugins → Add New → Upload Plugin
2. Activate
3. Open a post or page... click Set featured image... use the From URL tab
